Nâng cấp hệ thống đánh giá sản phẩm WooCommerce với tóm tắt xếp hạng, lọc/sắp xếp, và bình chọn hữu ích bằng AJAX.

== Description ==
Các tính năng chính:
- Hiển thị tóm tắt điểm đánh giá trung bình
- Biểu đồ phân bổ số lượng sao (star distribution)
- Lọc/sắp xếp đánh giá theo nhiều tiêu chí (mới nhất, cao nhất, thấp nhất, hữu ích nhất)
- Bình chọn mức độ hữu ích cho từng đánh giá
- Sử dụng AJAX, giao diện responsive, tương thích với WooCommerce

== Installation ==
1. Upload plugin vào thư mục /wp-content/plugins/.
2. Kích hoạt plugin qua trang quản trị WordPress.
